NSIS MUI (version 1) bitmaps are created with Gimp 2.0:
- use layered selection of bitmap wanted: install or uninstall
- File->Export As file extension .bmp
  * Options: Advanced : do not export colormap
  *          export as 24 bit (3 x 8) map
